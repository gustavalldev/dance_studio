# DanceStudio

Простая инструкция для запуска проекта. 

Ничего писать в коде не нужно. Достаточно установить 2 программы, открыть Терминал и по очереди вставить несколько команд.

## Что это за проект

Это локальный сайт для танцевальной школы.
После запуска он открывается в браузере на вашем компьютере и позволяет войти как:

- администратор
- преподаватель
- ученик

## Что нужно установить заранее

Перед запуском установите:

1. `Docker Desktop`
2. `Node.js` версии `20` или новее

Если после установки программы просят перезагрузить компьютер, лучше это сделать.

## Первый запуск

### 1. Откройте папку проекта

Убедитесь, что у вас уже есть папка `dancestudio` с файлами проекта.

### 2. Откройте Терминал в папке проекта

Если у вас macOS:

1. Откройте приложение `Terminal`
2. Напишите команду ниже, заменив путь на ваш, если папка лежит в другом месте:

```bash
cd /Users/kirill/Documents/project/dancestudio
```

Если не уверены, можно просто открыть папку проекта в Finder и перетащить ее в окно Терминала после команды `cd `.

### 3. Подготовьте файл настроек

Вставьте команду:

```bash
cp .env.example .env
```

Если Терминал напишет, что файл уже существует, ничего страшного. Просто переходите дальше.

### 4. Запустите базу данных

Сначала откройте `Docker Desktop` и дождитесь, пока он полностью запустится.

Потом вставьте команду:

```bash
docker compose up -d
```

Если все нормально, база данных запустится в фоне.

### 5. Установите зависимости проекта

Вставьте команду:

```bash
npm install
```

Это может занять несколько минут.

### 6. Создайте тестовые учетные записи

Вставьте команду:

```bash
npm run seed:demo
```

Эта команда создаст готовые логины для входа в систему.

### 7. Запустите сайт

Вставьте команду:

```bash
npm start
```

Если все запустилось успешно, в Терминале появится строка примерно такого вида:

```text
DanceStudio app listening on http://localhost:3001
```

## Как открыть сайт

Откройте браузер и перейдите по адресу:

`http://localhost:3001`

Также можно сразу открыть нужную роль:

- главная страница: `http://localhost:3001/`
- администратор: `http://localhost:3001/admin`
- преподаватель: `http://localhost:3001/teacher`
- ученик: `http://localhost:3001/student`

## Как остановить проект

Когда закончите работу:

### Остановить сайт

Перейдите в окно Терминала, где работает сайт, и нажмите:

```text
Control + C
```

### Остановить базу данных

В новом окне Терминала, находясь в папке проекта, выполните:

```bash
docker compose down
```

## Как запустить проект в следующий раз

Если проект уже запускался раньше, обычно достаточно этих шагов:

```bash
cd /Users/kirill/Documents/project/dancestudio
docker compose up -d
npm start
```

Если тестовые пользователи по какой-то причине пропали, повторно выполните:

```bash
npm run seed:demo
```

## Если что-то не работает

### Ошибка про `docker`

Скорее всего, не установлен `Docker Desktop` или он не запущен.

### Ошибка про `npm`

Скорее всего, не установлен `Node.js`.

### Сайт не открывается

Проверьте:

1. Открыт ли `Docker Desktop`
2. Выполняется ли команда `npm start`
3. Точно ли открыт адрес `http://localhost:3001`

### Порт занят

Если Терминал пишет, что порт `3001` занят, значит на компьютере уже работает что-то другое на этом адресе. В таком случае нужно закрыть старый процесс или поменять порт в файле `.env`.

## Важно

- Проект работает локально, только на этом компьютере
- Пока открыто окно с `npm start`, сайт работает
- Если закрыть Терминал, сайт остановится
